Message type

Ack

`Ack` acknowledges a previously received message. It copies the original `Id`, returns a `Result` code, and may include a short plain-text explanation in the element body.

Attribute Description Required
Id`Id` from the original message.Yes
ResultApplication result code.Yes
 
<Ack Id="123" Result="200">OK</Ack>

Message type

Heartbeat

`Heartbeat` is used to monitor the connection. Either side may send it. If an `Id` is present, the peer is expected to answer with an `Ack`.

Attribute Description Required
IdRequests an acknowledgement when present.No

With acknowledgement requested

<Heartbeat Id="123" />
<Ack Id="123" Result="200">OK</Ack>

Without acknowledgement requested

<Heartbeat />

Message type

TextMessage

`TextMessage` requests delivery of a plain text message.

Attribute Description Required
IdRequests an acknowledgement when present.No
ToRecipient number or address.No
FromSender number or address.No

The message body contains the text itself.

<TextMessage To="someone@somewhere.com">Hello,
This is a text message.</TextMessage>

Message type

Alarm

`Alarm` carries alarm or event data. The draft defines a compact core structure built around `Source`, `Type`, and optional `Location`.

Attribute Description Required
IdRequests an acknowledgement when present.No
Element Description Required
SourceIdentifies the source of the alarm.Yes
TypeIdentifies the alarm type and may carry additional qualifiers such as severity.Yes
LocationDescribes the location when it is known.No

Simple example

<Alarm><Source>209</Source><Type>FIRE</Type></Alarm>

Detailed example

<Alarm Id="101">
    <Source>
        <Section>5</Section>
        <Room>50</Room>
    </Source>
    <Type Qualifier="New" Severity="100">FIRE</Type>
    <Location>
        <Gps>
            <Longitude>17.87930</Longitude>
            <Latitude>59.41310</Latitude>
            <Altitude>57</Altitude>
        </Gps>
    </Location>
</Alarm>

This example is intentionally limited to fields that are described elsewhere in the current draft.
